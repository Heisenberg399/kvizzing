# KVizzing Fan Quiz

A web-based quiz app inspired by "KVizzing with Kumar Varun".

## Features
- Narrative-style questions with subtle hints
- Text input for answers (not MCQ)
- Smart answer checking (accepts close answers)
- Streak scoring
- Hint system (using a hint disables streak for that question)
- Questions stored in `questions.js` for easy editing

## Project Structure
```
/kvizzing_app
|-- index.html      # Main HTML file
|-- style.css       # Styling
|-- questions.js    # Question database
|-- script.js       # App logic
|-- README.md       # Setup instructions
```

## How to Run
1. Open this folder in VS Code.
2. Install the Live Server extension (by Ritwick Dey).
3. Right-click `index.html` and select "Open with Live Server".
4. Play the quiz!

You can add/edit questions in `questions.js`.

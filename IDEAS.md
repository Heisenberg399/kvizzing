# KVizzing — Feature Ideas

## 🎨 Polish & UX
- [ ] **Sound Effects Toggle** — Add a settings gear to mute/unmute audio
- [ ] **Theme Customization** — Make purchased store themes (Dark Mode) actually apply
- [ ] **Answer Reveal Animation** — Flashy animation when showing correct/incorrect answers

## 🏆 Competitive & Social
- [ ] **Weekly Leaderboard** — "This Week's Top Players" tab alongside All-Time leaderboard
- [ ] **Create Your Own Quiz** — Users submit questions that others can play (community content)

## 📅 Daily & Engagement
- [ ] **Daily Challenge** — Fixed 5 questions per day, same for everyone, separate daily leaderboard
- [ ] **Difficulty Levels** — Easy / Medium / Hard with point multipliers (1x / 2x / 3x)

## 🎖️ Progression
- [ ] **Achievement Badges** — "First Quiz", "10 Streak", "100 Points" etc. shown on profile
- [ ] **XP Levels & Ranks** — Bronze → Silver → Gold → Diamond with a progress bar
- [ ] **Question Categories** — Science, History, Sports, Movies etc. let users pick
